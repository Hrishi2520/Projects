# Features
* taking input from source.
* source like (string data, image data, audio data)
* it allows user to take inpute(type or write) in note application
* it should store the user provided note data in the database.
* we can archive the notes.
* we can delete the note.
* we can update or modify the data in existing note.
* we can share the stored note to another user.
* it should have search filter to find specific note.

# RequireMents

* language: Java 

* JDK: Java 19 jdk

* IDE: Intellij

* database: sql / in future will try with redis.
